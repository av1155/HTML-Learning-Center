Types of Web Storage Objects

  • SessionStorage()
  • LocalStorage()

Local vs. Session

  • Session Storage is destroyed once the user closes the browser
  • Local Storage stores data with no expiration date

Working with Values

The syntax for web storage for both local and session storage is very simple and similar. The data is stored as key/value pairs.

There are four different attributes for storing data

  • Storing a value:
    • .setItem
    • An example would be:
      • localStorage.setItem("key1", "value1");

  • Getting a value:
    • .getItem
    • An example would be:
      • alert(localStorage.getItem("key1"));

  • Removing a value:
    • .removeItem
    • An example would be:
      • localStorage.removeItem("key1");

  • Removing all values:
    • .clear
    • An example would be:
      • localStorage.clear();

Scalable Vector Graphics

SVG is used to draw shapes with HTML-style markup.

An SVG can be inserted to HTML code with a basic img tag and a src attribute pointing to the image.

<img src="image.svg" alt="" height="100">

Circle

To create a circle a svg tag should be used.

  • <svg width="150" height="150">
    • <circle cx="80" cy="80" r="50" fill="orange"
  • </svg>

You can also do other shapes such as rectangles, lines, polylines, ellipses, and polygons with similar code.

SVG Animations

SVG animations can be created by using the <animate> element.

  • attributeName:Specifies which attribute will be affected by the animation.
  • from:Specifies the starting value of the attribute.
  • to:Specifies the ending value of the attribute.
  • dur:Specifies how long the animation runs (duration).
  • fill:Specifies whether or not the attribute's value should return to its initial value when the animation is finished (Values: "remove" resets the value; "freeze" keeps the “to value”).
  • repeatCount:Specifies the repeat count of the animation.

In the example above, the rectangle changes its x attribute from 0 to 500 in 3 seconds.

The <path> element is used to define a path.

  • M: moveto
  • L: lineto
  • H: horizontal lineto
  • V: vertical lineto
  • C: curveto
  • S: smooth curveto
  • Q: quadratic Bézier curve
  • T: smooth quadratic Bézier curveto
  • A: elliptical Arc
  • Z: closepath

M places our "virtual pen" down at the position 0, 0. It then moves 200px down and to the right, then moves up to the position 200, 0. The Z command closes the shape, which results in a hypotenuse.

Canvas Element

This is an element that is used to draw graphics. It is usually used in conjunction with JavaScript.

The Canvas ID attribute is used so it can be run by a script from JavaScript. It has two coordinates, X and Y, which are in the upper left corner of the canvas. X increases to the right, and Y increases towards the bottom. Canvas is only a containar for graphics.

Differences Between SVG and Canvas Elements

The main difference between the two is that Canvas is more versatile, whereas SVG is more specific; however, SVG has more accesibility. This means that the element canvas can be used on more situations than the SVG element.

Canvas Transformation

The element "translate" is used to move the canvas element.

The element "rotate" is used to rotate the canvas element.

The element "scale" is used to scale the current drawing.

HTML5 Forms

Forms in HTML5 are similar to the ones I already learnt; however, they include many benefits. There are new attributes and input types that were introduced to help create better experiences. It is still done the same way as before.


A new attribute called "placeholder" is used to provide the user a hint on what to input in the form. "Autofocus" is another element used to focus the form immediately. The "required" element is used to make the input element required.

A really interesting new feature is a search box. The "search" input is now available for this.

The "datalist" tag can be used to define a list of pre-defined options forr the search field.
