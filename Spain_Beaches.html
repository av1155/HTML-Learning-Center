Commands

There are many more commands for styling text, these are:
Bold "b"
Italics "i"
Underline "u"
Big Text "big"
Big text "through CSS by writing span, then style, then font-size, then larger."
Small Text "small"
This is used to bold and make it important for the user "strong"
This is used to italic and make it important for the user "em"
Supscript = Higher baseline for typographical reasons "sup"
Subscript = Lower baseline for typographical reasons "sub"
Text that has been inserted (update) to a document "ins"
Text that has been deleted (update) from a document "del"

Another command is the use of ordered lists "ol" and unordered lists "ul."
The actual list attribute is "li."
An example of a numbered list would be:

  1. Number 1
  2. Number 2
  3. Number 3
  4. Number 4
An example of an unordered list would be:
  • Number 1
